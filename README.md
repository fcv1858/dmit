## 说说DMIT是谁

DMIT这家公司2017年成立，据说是几个在美国留学的华人学生创立的。公司注册在美国纽约，主要做海外VPS生意。

他们家现在主要有三个机房：美国洛杉矶、中国香港和日本东京。硬件配置很给力，全系用的是AMD EPYC处理器，配上企业级SSD固态硬盘。

最关键的是网络线路。他们主打CN2 GIA高端优化线路,这条线路对于需要稳定访问国内的用户来说,简直是刚需。电信、联通、移动三网都有专门优化,晚高峰也不掉速。

支付方式也很友好,支持支付宝、微信、PayPal,国内用户购买没什么障碍。

## 产品线到底怎么分

第一次打开DMIT官网,可能会被一堆套餐名称搞晕。什么Pro、Eyeball、Tier 1,看着就头大。

其实很简单,他们就三个系列:

**Premium系列**(代号Pro):这是旗舰产品线,走的CN2 GIA或者AS9929这种高端线路。国内访问速度快,延迟低,就是价格稍微贵点。适合对网络质量要求高的用户。

**Eyeball系列**(代号EB):可以理解为性价比之选。线路用的CMIN2,质量也不差,价格比Pro便宜不少。对于预算有限但又想要不错体验的用户,这个系列挺合适。

**Tier 1系列**(代号T1):经济型方案,走普通国际线路。国内访问速度一般,但胜在便宜,而且流量给得很足。做一些不太在意延迟的业务完全够用。

每个系列在洛杉矶、香港、东京三个机房都有部署,可以根据自己的需求选择。

## 实际使用体验怎么样

### 网络速度

我自己用的是洛杉矶Pro套餐,三网回程都是CN2 GIA线路。平时测试下来,电信和联通的延迟在150-160ms左右,移动稍高一点在170ms上下。

晚高峰时段也很稳定,基本不会出现明显的速度下降。丢包率控制在0.1%以下,这个数据在同价位VPS里算是很优秀了。

有个朋友用的香港Pro套餐,延迟更低,基本在40-50ms。不过香港的价格确实贵,同样配置差不多是洛杉矶的两倍。

### 硬件性能

CPU用的AMD EPYC系列,单核性能很强。我的1核套餐跑Geekbench 6能到1700多分,比很多2核的VPS都要好。

硬盘是NVMe SSD,读写速度没得说。IO测试下来,读取能到100MB/s以上,写入也有60MB/s左右。

### 售后服务

DMIT的工单系统响应还算及时,一般24小时内会有回复。不过别指望能有多快,毕竟不是24小时在线客服。

有一次我配置SSL证书遇到问题,提了工单后20分钟就收到回复,技术支持不仅解决了问题,还给了详细的配置指南。这个体验确实不错。

IP被墙可以免费更换,基本规则是每15天可换一次。其他情况换IP要收5美元。这个政策对于某些特殊用途的用户来说还是挺友好的。

### 流媒体解锁

我测试了洛杉矶的IP,除了Netflix和Disney+不解锁,其他大部分流媒体都能正常访问,包括YouTube Premium。IP段识别为美国洛杉矶本地。

不过DMIT官方并不保证流媒体解锁,这个要看运气。如果特别在意解锁,建议购买前先问问客服。

## 哪些人适合用DMIT

说了这么多,总结一下DMIT到底适合什么样的用户:

**需要稳定建站的用户** - 如果你要搭建网站,特别是面向国内用户的网站,DMIT的CN2 GIA线路能保证访问速度。晚高峰不掉速这一点很关键。

**对网络质量有要求的开发者** - 做远程开发、API接口这类对延迟和稳定性要求高的场景,DMIT的表现确实不错。

**预算有一定余地的用户** - DMIT不算特别便宜,但也不是最贵的。如果你能接受年付100-300美元的预算,性价比还是可以的。

**不想频繁换VPS的用户** - DMIT的稳定性确实好,基本不会出现那种突然连不上或者频繁维护的情况。买了就能踏实用,不用三天两头操心。

反过来说,如果你只是想练手玩玩,或者预算特别紧张,可以考虑他们家的👉 [Tier 1系列](https://www.dmit.io/aff.php?aff=13832&gid=11),最便宜的年付才36美元。虽然没有CN2优化,但做一些不太在意延迟的业务完全够用。

## 一些使用建议

根据我的使用经验,给几个实用建议:

**先从小套餐开始** - 不确定是否适合自己,可以先买最低配测试一个月。DMIT支持套餐升级,确认满意后再加配置。

**年付更划算** - 如果确定要长期用,直接年付能省不少钱。很多优惠码只对季付以上生效,而且年付通常有额外折扣。

**关注限量版套餐** - DMIT经常会推出一些限量版特价套餐,性价比很高。不过这类套餐补货就秒光,要经常关注才能抢到。

**合理选择机房** - 如果主要服务国内用户,香港延迟最低但价格贵;洛杉矶性价比最好;东京介于两者之间。根据预算和需求来选。

**注意流量限制** - 虽然DMIT流量用完后只是限速不停机,但还是要注意自己的使用量。如果经常跑满带宽,建议选择流量更大的套餐。
